Configuración de Slack

Usuarios de Slack

Agrega los IDs de usuarios de Slack que recibirán notificaciones cuando se cree un nuevo ticket. Puedes agregar múltiples usuarios.

¿Cómo obtener el User ID de Slack?
Ve al perfil del usuario en Slack y copia el ID de la URL, o úsalo directamente en un mensaje directo. El formato típico es algo como U06L3SJANAD.

Mensaje Personalizado

Personaliza el mensaje que se enviará a Slack cuando se cree un ticket. Puedes usar variables para incluir información dinámica del ticket.

Puedes usar variables como {folio}, {solicitante}, {prioridad}, {categoria}, {descripcion}, etc.

Variables disponibles:

  • {folio} - Número de folio del ticket
  • {ticketId} - ID del ticket
  • {solicitante} - Nombre del solicitante
  • {email} - Email del solicitante
  • {prioridad} - Prioridad del ticket
  • {categoria} - Categoría del ticket
  • {descripcion} - Descripción del ticket
  • {maquina} - Máquina relacionada (si aplica)
  • {elemento} - Elemento relacionado (si aplica)
  • {url} - URL para ver el ticket

Si dejas este campo vacío, se usará el mensaje predeterminado del sistema.
